# Kelza Operations Playbook

> "Products create users.
>
> Operations create companies."

---

# Purpose

This document defines how Kelza operates after launch.

Software alone does not create a successful marketplace.

Healthy operations do.

This playbook standardizes how Kelza recruits creators, supports communities, manages trust, responds to incidents, expands internationally and continuously improves the ecosystem.

Every operational decision should reinforce one objective:

Create the safest, healthiest and most trusted global relationship marketplace.

---

# Operating Principles

Every team follows these principles.

People before metrics.

Quality before quantity.

Communities before downloads.

Trust before growth.

Long-term thinking over short-term optimization.

Creator success equals company success.

Healthy communities create sustainable business.
